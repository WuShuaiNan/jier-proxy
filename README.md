# JIER Proxy Introduction - JIER 网络代理服务 简介
- JIER Proxy 是一款开箱即用的网络代理服务。

## 1. 项目功能
- 正向代理
- 反向代理
- 内网穿透
- P2P点对点加密通信
- 虚拟网络
- DNS、HTTP劫持修改
- Unix套接字转发
- 文件访问服务
- 自定义域名访问
- HTTP路由
- 网络代理手动负载均衡
- HTTP转HTTPS
